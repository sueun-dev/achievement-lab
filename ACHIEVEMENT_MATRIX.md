## Natural Execution Rules

- Do not create synthetic repos purely for badge farming.
- Tie every PR/discussion to a real project outcome.
- Prefer repeatable weekly contribution habits over one-off actions.

## 90-Day Plan

1. External contributions:
   - Submit 2+ PRs/week to active open-source repositories.
   - Target repos where maintainers merge quickly and discussions are active.
2. Discussion quality:
   - Post practical answers in Q&A discussions with reproducible steps.
   - Aim for accepted answers via clear troubleshooting structure.
3. Flagship repo growth:
   - Pick one project, publish release notes, add examples, and write usage docs.
   - Share updates consistently to grow to first `16` stars.
4. Collaboration depth:
   - Pair with at least one collaborator each month on merged PRs.
   - Keep co-authored work tied to real features or bugfixes.
